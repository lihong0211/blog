# [561. 数组拆分 I](https://leetcode.cn/problems/array-partition-i/)

> 原题链接：[LeetCode 561. 数组拆分 I](https://leetcode.cn/problems/array-partition-i/)

## 题解代码

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var arrayPairSum = function(nums) {
  nums = nums.sort((m, n) => m - n)
  let res = 0
  for (let i = 0; i < nums.length; i++) {
      if (!(i % 2)) res += nums[i]
  }
  return res
};
```
