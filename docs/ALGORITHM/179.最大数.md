## 题目描述

给定一组非负整数 `nums`，重新排列每个数的顺序（每个数不可拆分）使之组成一个最大的整数。

**注意：**输出结果可能非常大，所以你需要返回一个字符串而不是整数。

- `1 <= nums.length <= 100`

- 0 <= nums`[i]` <= 109

**难度：** Medium

## 题解代码

```javascript
/**
 * @param {number[]} nums
 * @return {string}
 */
const largestNumber = nums => {
    nums = nums.sort((a, b) => {
        const s1 = `${a}${b}`
        const s2 = `${b}${a}`
        return s2 - s1
    })
    return nums[0] ? nums.join('') : '0'
};
```
