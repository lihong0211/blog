# [34. 在排序数组中查找元素的第一个和最后一个位置](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/)

> 原题链接：[LeetCode 34. 在排序数组中查找元素的第一个和最后一个位置](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/)








## 题目描述

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

 

**示例 1：**

```
5,7,7,8,8,10]
```

**示例 2：**

```
5,7,7,8,8,10]
```

**示例 3：**

- **输入：** nums = `[]`, target = 0
- **输出：** `[-1,-1]`

 

**提示：**

- 0 <= nums.length <= 105
	-109 <= nums`[i]` <= 109
- `nums` 是一个非递减数组

	-109 <= target <= 109

**难度：** Medium

---

## 题解代码

```javascript
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var searchRange = function(nums, target) {
  const p = find(nums, 0, nums.length - 1, target)
  if (p === -1) return [-1, -1]
  let i = p, j = p
  while(nums[i] === nums[p]) {
    i--
  }
  while(nums[j] === nums[p]) {
    j++
  }
  return [i + 1, j - 1]
};

function find (nums, l, r, target) {
  if (l > r) return -1
  let mid = Math.floor(l + (r - l) / 2)
  if (nums[mid] < target) return find(nums, mid + 1, r, target)
  if (nums[mid] > target) return find(nums, l, mid - 1, target)
  return mid
}
```
